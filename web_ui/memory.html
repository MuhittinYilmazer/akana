          Settings

          Memory behavior, vector search and storage status.

          Memory behavior

          When on, Akana writes what you ask it to «remember» directly to permanent facts. When off, everything goes to Inbox first and waits for your approval.

          When on, Akana summarizes your idle or long conversations into the Inbox for you to review and keep. When off, no session summaries are created automatically.

          Vector (semantic) search

          Local: multilingual model, downloads ~220MB on first use, runs in-process (no Ollama needed). The Ollama fields below are only used when «Ollama» is selected.

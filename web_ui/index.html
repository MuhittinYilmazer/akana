    Tailnet address

    How Tailscale works & manual setup

    Akana runs locally on this machine — it is not published to the public internet. Tailscale is a private, encrypted network (a personal VPN) that lets your phone and other devices reach this machine directly, as if they were on the same home network — no port-forwarding or exposing the server.

    One-time setup:

    1. Install Tailscale on this machine and on your phone, then sign in to the same account on both.
    2. Find this machine's Tailscale address — its name (e.g. your-host.tailnet.ts.net) or its 100.x.y.z IP, shown in the Tailscale app.
    3. On the phone's browser open http://<tailscale-address>:<port> (the port is shown in the effective endpoint above).
    4. When asked to authenticate, enter the API token from the field above.
    5. Tip: from the browser menu choose Add to Home Screen so Akana opens like an app.

    Prefer not to type the token on the phone? Use Connect your phone below to scan a QR that loads it automatically.

    Connect phone

    «Hey Akana» wake word

    Loading status…

    Automatically starts Hey Akana listening in the top bar when the page opens. If off, it only resumes after a page reload.

    Model uses the on-server «Hey Akana» detector (gated by the threshold below). Browser uses your browser's speech recognition to match the «Hey Akana» phrase. If the model isn't set up, it falls back to browser automatically.

    Applies to Model mode only. Low threshold = more sensitive (more false positives). Live score is shown in the top bar.

    Applies to Model mode only. The score must stay above the threshold for this many consecutive frames (~80 ms each) before it triggers. Higher = stricter, far fewer false wakes; 1 = fire on a single peak. Use this when max threshold still over-detects.

    Conversational voice mode

    Hands-free conversation — Akana listens, responds, and goes back to listening when you pause. These settings are stored in the browser and apply to the next conversation.

    Browser uses your browser's built-in speech recognition and detects when you stop speaking on its own. Whisper records your turn and transcribes it on the server for one consistent engine — it needs the voice extra installed (faster-whisper). Applies to the next conversation.

    When on, the wake word directly starts hands-free conversation. When off, only a single command is taken.

    Off by default. When on, if you start speaking while Akana is talking it interrupts and listens to you (can self-trigger on speakers — raise the threshold below if so).

    The voice threshold your voice must exceed to interrupt Akana. Low = easier/more sensitive barge-in; high = more noise-resistant (but you need to speak louder). Too low and Akana may trigger on its own voice. (Typical: your voice ~0.04, ambient ~0.005.)

    Short sounds play when listening starts/stops. Default off.

    How long of silence is assumed to mean you have finished speaking. Short = faster response, long = less likely to cut you off.

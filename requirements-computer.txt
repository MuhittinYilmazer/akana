# Computer-control pack (akana/computer-control) — screenshot + mouse + keyboard.

# These power the `computer` MCP server (akana_server/computer_mcp). They are an
# OPTIONAL extra: Akana runs fine without them; only the computer.* tools need them.
# Install into Akana's own environment (the venv), NOT user-site.
#
# TRAP (this machine forces PIP_USER): a bare `pip install` lands in user-site and the
# venv never sees it. Install with either:
#     PIP_USER=0 venv/Scripts/pip install --no-user -r requirements-computer.txt
# or clear PIP_USER in the environment first. Verify with:
#     venv/Scripts/python -c "import mss, pyautogui, pygetwindow"
#
# Pillow is already a core Akana dependency (screenshot PNG encoding uses it).
